# Hotel Senator Website

React/Vite concept website for The Hotel Senator in Saskatoon, including connected pages for Winston's English Pub, Flanagan's Steakhouse, and 21st Street Brewery.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Vercel settings

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`
